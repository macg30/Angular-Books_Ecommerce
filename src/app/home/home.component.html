¡Bienvenido! Esta app tiene como objetivo

Facilitar las gestión de libros relacionados con el framework de Angular mediante el uso de formularios reactivos y el uso de LocalStorage como estrategia de persistencia. Además utiliza un web service de terceros para obtener el importe equivalente con respecto al EUR, según la tasa de cambio actual.
